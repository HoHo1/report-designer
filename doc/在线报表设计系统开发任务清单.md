## 十、使用说明

### 状态符号说明
- 🔄 进行中：任务正在执行
- ✅ 已完成：任务已完成
- ❌ 已取消：任务已取消

### 部署说明
1. 执行 `sql/init.sql` 初始化数据库
2. 修改 `application.yml` 中的数据库和 Redis 连接配置
3. 使用 Maven 构建后端：`mvn clean package`
4. 使用 Docker Compose 启动：`docker-compose up -d`

### 访问地址
- 前端应用：http://localhost:80
- 后端接口：http://localhost:8080
- API 文档：http://localhost:8080/doc.html
- 默认账号：admin / admin123
